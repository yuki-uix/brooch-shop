# T4.1: Chat Panel UI（侧栏容器 + 触发按钮）

## Summary

在全局布局中注入可折叠的购物助手侧栏（ShoppingChatPanel），用户点击触发按钮后从右侧滑入，不遮挡主内容区；移动端全屏展示。本任务仅负责 UI 容器层，不涉及 AI 接口调用。

## PM

- **里程碑**: M5
- **依赖**: 无（可独立开发，T4.2 完成后接入真实 API）
- **风险 / 备注**: R6（移动端键盘弹出遮挡输入框），需响应 `visualViewport` resize 调整底部 padding。

## BA — 用户故事

**US-4.1** 作为 **用户**，我希望在购物页随时打开 AI 购物助手侧栏，以便在不离开商品列表的情况下获得帮助。

### 验收标准

- [x] 导航栏或右下角有「AI 助手」触发按钮
- [x] 点击后从右侧滑入 Chat Panel，不遮挡主内容区域
- [x] 可随时关闭，关闭后对话历史本次 session 内保留
- [x] 移动端 Panel 全屏展示

## UX

- **相关页面 / 流程**: 全局布局（所有页面右下角均可触发）
- **线框 / 布局**:

```
桌面端（Panel 展开）:
+-----------------------------+------------------+
| 主内容区（商品列表等）       |  AI 助手         |
|                             | +--------------+ |
|                             | | 🤖 你好！     | |
|                             | | 有什么可以帮  | |
|                             | | 你的？        | |
|                             | +--------------+ |
|                             | [输入框] [发送] |
+-----------------------------+------------------+
                                  [×关闭]

移动端（Panel 全屏）:
+----------------------------------+
| [← 返回]         AI 助手         |
|----------------------------------|
| 🤖 你好！有什么可以帮你的？       |
|                                  |
|                                  |
|----------------------------------|
| [输入框]                  [发送] |
+----------------------------------+

触发按钮（右下角 FAB）:
       [💬 AI 助手]
```

- **令牌 / 组件**: `ShoppingChatPanel`（新建）, `ChatMessage`（新建）, FAB 按钮使用主题 `lavender-600`，Panel 宽度桌面端 `w-96`，移动端 `w-full h-full`

## TL — 技术

- **ADR 引用**: ADR-6（useChat in-memory 对话历史）
- **实现备注**:
  - `app/components/ShoppingChatPanel.tsx` — 侧栏容器，管理 open/close 状态；使用 `useChat` hook（`api: '/api/chat/shopping'`）
  - `app/components/ChatMessage.tsx` — 单条消息渲染（role: user / assistant），本 task 先渲染纯文本，T4.4 再扩展卡片
  - `app/layout.tsx` — 在根布局注入 `<ShoppingChatPanel />`，确保全局可用
  - Panel 滑入动画: Tailwind `transition-transform translate-x-full / translate-x-0`
  - 移动端键盘处理: 监听 `window.visualViewport?.addEventListener('resize', ...)` 动态设置 `paddingBottom`
- **不纳入**: AI 接口调用（T4.2）；商品卡片渲染（T4.4）；addToCart 工具（T4.3）

## QA

- **TCs**: TC-4.1
- **边界用例**:
  - 快速连续点击触发按钮，Panel 不应多次叠加渲染
  - 关闭 Panel 后重新打开，messages 数组保留（session 内）
  - 移动端横屏切换时 Panel 高度自适应
- **质量门禁**: Panel 在桌面/移动双端可正常开关，动画无闪烁
