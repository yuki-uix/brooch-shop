# T0.2: 商品列表页与材质筛选

## Summary

实现商品列表页，支持按材质筛选，每张商品卡片展示材质标签，用户可一键清除筛选。

## PM

- **里程碑**: M1
- **依赖**: T0.1（数据模型就绪）
- **风险 / 备注**: 筛选逻辑需与首页材质入口保持参数一致。

## BA — 用户故事

**US-0.2** 作为 **用户**，我希望在商品列表页按材质筛选胸针，以便快速找到我喜欢的材质类型。

### 验收标准

- [x] 列表页显示材质筛选入口（标签/下拉）
- [x] 选择材质后，列表仅展示对应商品
- [x] 可清除筛选恢复全部商品
- [x] 每件商品卡片上显示材质标签
