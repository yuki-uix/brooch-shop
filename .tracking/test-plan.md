### 边界 / 异常用例

- 上传非图片文件（如 .pdf / .doc）→ 应拒绝或提示格式不支持
- 文字描述超过 500 字 → 应截断或提示
- AI API 超时（>10s）→ 友好错误提示，可重试
- AI 返回非法结构 → 兜底错误提示
- 描述含极端内容（违禁意象）→ 返回「需确认」或拒绝
- 购物车为空时点击「去结账」→ 应禁用或提示
- 结账时必填项为空 → 前端校验拦截
