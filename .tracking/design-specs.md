# 设计说明

## 设计原则

- **材质即导航**: 材质是胸针选购的核心维度，在首页、列表、商详都保持强存在感
- **定制三步闭环**: 输入 → 结果 → 结账，不超过 3 步完成定制流程
- **透明可信**: AI 报价结果附带简短理由与免责声明，避免用户误解
- **一致性**: 定制行与标准品在购物车/结账中视觉一致，仅用标签区分类型

## 用户流程

### Flow: 标准品购买

1. 首页/类目页 → 选择材质入口或浏览全部
2. 列表页 → 按材质筛选 → 点击商品卡片
3. 商详页 → 查看信息 → 「加入购物车」
4. 购物车 → 调整数量 → 「去结账」
5. 结账页 → 填写收货信息 → 「提交订单」
6. 订单确认页

### Flow: 定制询价

1. 任意页导航 → 点击「定制」入口
2. 定制页 → 上传图片 和/或 输入描述 → 「提交」
3. Loading → 结果页：可行性标签 + 理由 + 报价 + 免责声明
4. 「可制作」→ 「加入购物车」/「立即结账」
5. 购物车/结账 → 与标准品统一流程

## 组件 / 模式

- **MaterialTag**: 材质标签组件，复用于卡片、详情、筛选
- **ProductCard**: 商品卡片（图 + 名 + 材质标签 + 价格）
- **CartItem**: 购物车行项目（标准品/定制品两种变体）
- **FeasibilityBadge**: 可行性状态标签（可制作 / 需确认 / 不建议），分别对应 success / warning / danger 色
- **FileUploader**: 图片上传组件（拖拽 + 点击，预览 + 删除）
- **PriceSnapshot**: 报价快照展示（金额 + 免责声明）
